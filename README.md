# `_why_the_lucky_markov`
A markov chain generator based on the various works of `_why the lucky stiff.`

[Checkout the open "Issues" and vote for your favorites](https://github.com/rossfuhrman/_why_the_lucky_markov/issues)

[https://why-the-lucky-markov.herokuapp.com](https://why-the-lucky-markov.herokuapp.com)

### Markov chain generated hourly
##### A new Toot will be generated every hour. A GitHub Issue will be created and a Toot will be posted at the below [Mastodon](https://botsin.space/@_why_the_lucky_markov) account. Later, an IFTTT trigger will pick that up and post it to the below [Twitter](https://twitter.com/noisy_why) account.

[Mastodon](https://botsin.space/@_why_the_lucky_markov)

[Twitter](https://twitter.com/noisy_why)

### Best-of generated Markov chains
##### Every hour, the comments on open GitHub Issues will be reviewed and any of them with enough upvotes will be posted to the below [Mastodon](https://mastodon.xyz/@_why_toots) account and the Issue will be closed. Later, an IFTTT trigger will pick that up and post it to the below [Twitter](https://twitter.com/tootswhy) account.

[Mastodon](https://mastodon.xyz/@_why_toots)

[Twitter](https://twitter.com/tootswhy)
